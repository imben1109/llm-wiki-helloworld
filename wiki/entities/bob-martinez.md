# Bob Martinez

Senior Backend Engineer who supported the [[llm-wiki-pattern]] pilot and volunteered to migrate post-mortems as the first batch of content.

## Overview

Bob Martinez is a Senior Backend Engineer on the same team as [[alice-chen]]. Initially sceptical — asking how the LLM Wiki Pattern differed from existing tools like Confluence — he was convinced by the automatic cross-referencing and contradiction detection features. He took on responsibility for migrating the team's last five post-mortems into `raw/notes/` as the seed content for the pilot.

## Key Points

- Raised the practical question of maintenance: who owns the schema/prompt file if it drifts.
- Identified scattered post-mortems (Notion, GitHub Issues, Google Docs) as the highest-value consolidation target.
- Assigned to migrate the first five post-mortems into the raw folder.
