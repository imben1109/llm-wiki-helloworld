# Outputs

This folder contains query results, generated reports, and analysis briefs.

Files here are written by the LLM in response to queries and `lint` reports.
They are for human consumption — you can read, copy, share, or delete them freely.

## Examples of what you will find here

- `query-YYYY-MM-DD-your-question.md` — Answer to a question about the wiki
- `lint-report-YYYY-MM-DD.md` — Health check report
- `synthesis-topic-name.md` — Deep-dive analysis of a topic
